## Deployment Flow

1. **Push code** to GitHub
2. **GitHub Actions** builds Docker images
3. **SSH to Proxmox** container
4. **Pull latest code** and rebuild containers
5. **Cloudflared** routes traffic through tunnel

See individual docs for detailed instructions.
